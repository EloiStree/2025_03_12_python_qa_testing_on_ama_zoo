> Proposal for a QA testing exercise designed to teach Python to beginners.

https://github.com/EloiStree/2025_03_12_godot_qa_testing_on_ama_zoo/  


---------

**Exercice FR:**

Le client AmaZoo doit vérifier, à chaque publication sur Steam, la qualité des traductions de son jeu dans toutes les langues disponibles.
Au minimum, le menu principal ainsi que les trois premiers niveaux doivent être vérifiés dans chaque langue.

Vous disposez d’une semaine pour développer un script Python capable de prendre automatiquement des captures d’écran de toutes les fenêtres du jeu dans toutes les langues :
https://store.steampowered.com/app/3066360/AmaZoo/

Aucune intervention humaine ne doit être nécessaire une fois le script lancé.


----------

**Exercise EN:**

The client AmaZoo must verify, for each publication on Steam, the quality of the translations of its game in all available languages.
At a minimum, the main menu and the first three levels must be checked in every language.

You have one week to develop a Python script capable of automatically taking screenshots of all game windows in every language:
https://store.steampowered.com/app/3066360/AmaZoo/

No human intervention should be required once the script is launched.

* Apprendre les bases du **langage Python**
